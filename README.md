# Atividade-PW-05-05
# Sistema de Cadastro

Sistema web responsivo para gerenciamento de Clientes, Categorias, Fornecedores e Funcionários, com armazenamento de dados em tempo real.

* **Nome:** Caio Candiani Regio
* **Turma:** 2DS

## Módulos
* **Home (`index.html`):** Tela inicial de boas-vindas.
* **Clientes (`clientes.html`):** Cadastro de Nome e E-mail.
* **Categorias (`categorias.html`):** Cadastro de Nome e Informações.
* **Fornecedores (`fornecedores.html`):** Cadastro de Nome, CNPJ, E-mail e Estado (SP, RJ, MG).
* **Funcionários (`funcionarios.html`):** Cadastro de Nome, E-mail e Cargo.
