# Sistema de Cadastro

Sistema web responsivo para gerenciamento de Clientes e Funcionários, com armazenamento de dados em tempo real.

* **Nome:** Caio Candiani Regio, Enzo Miguel Bertoluci
* **Turma:** 2DS

## Módulos
* **Home (`index.html`):** Tela inicial de boas-vindas, explica um pouco de como o site funciona.
* **Clientes (`clientes.html`):** Cadastro de Nome e E-mail, com a tabela de cadastros abaixo onde mostra os cadastros já feitos.
* **Funcionários (`Vendedores.html`):** Cadastro de Nome, Salario e Cargo, com a tabela de cadastros abaixo onde mostra os cadastros já feitos.
